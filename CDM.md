# CDM Overview

CDM - Customer Data Management

> Retain + Grow the Customer Base

- Process: 1. Collect 2. Store 3. Organize 4. Utilise to build a comprehensive Customer Profile
- Create a Single Consistent and view of Each Customer

### Types of Customer Data

1. Identity Data:

   - Basic personal information
   - Eg. Sign-up Forms, Discount Vouchers, Pre-Orders and Etc

2. Quantitative Data:

   - Customer Interactions and Transactions
   - Eg. Transactions: Purchased, Amounts, Order and Etc
   - Tools: Web Analytics, Cookies and Etc

3. Descriptive Data:

   - Demographic + Lifestyle Information
   - Eg. Family: Marital Status, Relationships, Property Type, Car, Pets, Hobbies, Education and Etc
   - Methods: Interviews, Surveys and Etc

### Data Governance

1. Audit Trails: Track Changes, Key Statistics and Data Usage
2. Process: Define Data Storage, Flow and System Interactions
3. Policy: Rules for Data Quality, Collection, Management and Privacy
