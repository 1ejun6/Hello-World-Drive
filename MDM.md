# MDM Overview

MDM - Master Data Management

> Create a single source of Truth: 1. Consistent 2. Accurate 3. Accessible

- Manage the Critical Data Entities; Eg. Customers, Products, Employees and Etc
- Golden Record: Unified and Authoritative view of a Customer
- Data Consistency and Accuracy: Avoid Discrepancies and Duplicates by using MDM across different systems
- Data Governance: Establish Rules and Policies for 1. Data Quality 2. Access 3. Security 4. Compliance
